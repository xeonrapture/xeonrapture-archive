# Intro

The standard calendar is based on the [[Xeon#The Xenic Seasons|xenic seasons]] which are the same across the whole supercluster.

# Calendar

There are 12 months per year, 5 weeks per month, and 6 days per week in the standard calendar.

## Months

| Season | Month      |
| ------ | ---------- |
| Spring | Uthalanai  |
|        | Ulaganai   |
|        | Zolanai    |
| Summer | Reinanai   |
|        | Iaianai    |
|        | Vorashanai |
| Fall   | Xozanai    |
|        | Qolanai    |
|        | Dodatanai  |
| Winter | Ethinanai  |
|        | Valkanai   |
|        | Karoshanai |

There's also a week after the end of Karoshanai called Abaiah that's a week full of celebrations! It's usually five days long, though sometimes it's 6.
