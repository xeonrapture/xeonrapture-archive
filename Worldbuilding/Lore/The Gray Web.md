The gray web is a twisting system of nodes built on top of the [[xr/Worldbuilding/Lore/The Xenic Plane|xenic plane]], a massive network of gray xeon that connects the entire supercluster. Each node is a self-contained little world with its own rules, used to store knowledge or credits, hold a whole constructed world, or network and search other nodes. Some are even used as living spaces for those who have decided to leave the physical world behind or have found themselves trapped in the gray web.

Nodes that connect lots of other nodes are called nexuses and usually provide some sort of search engine or database of links to make it easier for users to find the node they want.

The gray web is accessed through a terminal, or a xenic computer with gray web connectivity built in, and one can either search the information contained in nodes from outside through a terminal screen or run the web itself by hooking your brain into the network (called [[Webrunning 2.0|webrunning]]). More data and opportunities are available to webrunners than those just surfing from the outside, though the risks are also greater.

Not every node is easily accessible! Some node clusters are protected by digital security, some are really hard to find a route into, and others are remnants of the Old Web and contain hidden knowledge and unknown dangers.

The gray web was discovered by Valdan scientists, and nobody really knows who created it and when. The web technically exists in the [[xr/Worldbuilding/Lore/The Xenic Plane|xenic plane]] but is bound by its own rules and doesn't react to belief the way the xenic plane does; it is possible to find portals from the web into the xenic plane but these are very rare.
