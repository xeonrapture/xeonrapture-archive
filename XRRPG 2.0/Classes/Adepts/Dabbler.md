# Intro

Adepts channel the xeon through their own bodies to gain powers and enact potent effects.

Dabblers, however, have clear chromas, so they're not able to channel the xeon through their bodies like people with colored chromas do.

But because they aren't locked into a specialization from an early age they're able to learn lots of different skills from lots of different places, giving them a viewpoint and set of capabilities that other adepts don't.

When you level up your powers are *replaced* with the new tiers.

| Character Level | Class Level | Class Abilities                               |
| --------------- | ----------- | --------------------------------------------- |
| 0               | 0           | Basic Power Tier 0                            |
| 1               | 1           | Basic Power Tier 1 &<br>Advanced Power Tier 1 |
| 4               | 2           | Basic Power Tier 2 &<br>Advanced Power Tier 2 |
| 7               | 3           | Basic Power Tier 3 &<br>Advanced Power Tier 3 |
| 10              | 4           | Basic Power Tier 4                            |
